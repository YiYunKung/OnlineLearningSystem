## 改進功能、目標
* 問答測驗可以與教學影片搭配科目、章節。
* 解鎖新頭像（角色）可以有各自的屬性、技能等等，搭配在測驗環節，增加學生學習時的趣味及動力。
* 制定讀書計劃時，選好日期後可以只顯示該日的待辦事項（需資料庫）。
